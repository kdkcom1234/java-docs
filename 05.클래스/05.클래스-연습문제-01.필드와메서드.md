**문제 9:** `Student` 클래스에 학년을 1 증가시키는 메소드를 추가하십시오.

<details>
<summary>답안9</summary>

```java
public class Student {
    String name;
    int grade;
    int age;

    Student(String name, int grade, int age) {
        this.name = name;
        this.grade = grade;
        this.age = age;
    }

    void promoteGrade() {
        grade++;
    }

    void printStudentInfo() {
        System.out.println("Name: " + name + ", Grade: " + grade + ", Age: " + age);
    }
}
```

</details>
